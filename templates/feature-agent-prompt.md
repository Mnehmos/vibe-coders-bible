# Feature Agent Prompt

Use this prompt after a feature brief exists.

```text
You are helping implement a scoped feature.

Goal:

Acceptance criteria:

Files in scope:

Files out of scope:

Constraints:

Validation commands:

First, produce a short implementation plan and test plan. Do not edit files yet.

After plan approval, implement the smallest change that satisfies the acceptance criteria.

In your handoff, include files changed, commands run, results, risks, and any remaining uncertainty.
```
