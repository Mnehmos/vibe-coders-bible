# Output Targets

## PDF

Purpose:

- Stable visual layout.
- Printable review copy.
- Public downloadable edition.

Needs:

- Title page.
- Copyright and license page.
- Table of contents.
- Page numbers.
- Internal links.
- Tables that fit page width.
- Code blocks with readable wrapping.

## DOCX

Purpose:

- Editorial review.
- Copyediting.
- Publisher handoff.

Needs:

- Style-based headings.
- Comments and tracked changes support.
- Tables preserved.
- Code blocks legible.

## EPUB

Purpose:

- Ebook readers and stores.

Needs:

- Clean heading hierarchy.
- Metadata from `manuscript/metadata.yml`.
- Internal table of contents.
- Accessible alt text for images and diagrams.
- Tables converted or simplified when needed.

## Audiobook

Purpose:

- Spoken edition.

Needs:

- Narration script pass.
- Pronunciation guide.
- Table-to-prose conversions.
- Code block summaries.
- Chapter-level audio files.
- AI voice disclosure.
- Audio QA checklist.
