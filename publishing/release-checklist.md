# Release Checklist

## Manuscript

- [ ] All chapters expanded beyond scaffold.
- [ ] Front matter complete.
- [ ] Conclusion complete.
- [ ] Appendices complete.
- [ ] Chapter titles match `manuscript/MANIFEST.md`.
- [ ] Internal links checked.
- [ ] License page checked.

## Print and ebook

- [ ] PDF generated.
- [ ] DOCX generated.
- [ ] EPUB generated.
- [ ] Table of contents checked.
- [ ] Tables render correctly.
- [ ] Code blocks wrap legibly.
- [ ] Metadata correct.

## Audiobook

- [ ] Narration scripts prepared.
- [ ] Pronunciation guide complete.
- [ ] TTS model and voice selected.
- [ ] AI voice disclosure included.
- [ ] Chapter audio generated.
- [ ] Audio proof listened through.
- [ ] Mastered files exported.
- [ ] Runtime and file list recorded.

## Release

- [ ] Version tag created.
- [ ] Checksums generated.
- [ ] Release notes written.
- [ ] Artifacts uploaded.
- [ ] Source archive attached or linked.
