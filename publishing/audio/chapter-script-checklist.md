# Chapter Script Checklist

Use this before generating audio for any chapter.

- [ ] Chapter title matches `manuscript/MANIFEST.md`.
- [ ] Thesis is present.
- [ ] Key line is present.
- [ ] Tables are converted to narration-friendly prose.
- [ ] Long code blocks are summarized.
- [ ] File paths are spoken only when useful.
- [ ] Acronyms are expanded on first use.
- [ ] Product names have pronunciation notes when needed.
- [ ] Markdown artifacts that sound awkward are removed.
- [ ] Ending transition is natural.
- [ ] AI voice disclosure is included in audiobook front matter.
