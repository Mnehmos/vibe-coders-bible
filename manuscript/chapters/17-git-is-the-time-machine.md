# Chapter 17 - Git Is The Time Machine

Part: IV - The Propose / Validate / Commit Loop

## Thesis

AI makes changes fast; Git makes them survivable.

## Key line

If AI accelerates change, version control becomes more important, not less.

## Chapter job

Show how branches, commits, PRs, and reverts become higher-value controls in agentic development.

## Main beats

- Small commits.
- Branches.
- PRs.
- Reverts.
- Squash merges.
- Bisect.
- Tags.
- Release branches.
- Diff review.
- Commit messages as memory.

## Practical artifact

Add a "commit hygiene checklist":

- One reason for the commit.
- Intended files only.
- Tests or evidence included.
- Message names project behavior.
- Revert path is understandable.

## Draft notes

Tie commit history to trace. The repo should explain what became true and why.
