# Chapter 40 - Definition Of Done For AI-Assisted Work

Part: VIII - The Field Manual

## Thesis

AI-assisted work is not done when the model stops typing. It is done when the system, human, and trace agree that the change is safe to accept.

## Key line

Done means the change is built, verified, documented, and understood.

## Definition of done

- [ ] Code builds.
- [ ] Tests pass.
- [ ] New behavior is tested.
- [ ] Negative case is tested when relevant.
- [ ] Docs are updated.
- [ ] No secrets are exposed.
- [ ] Migration is reversible when relevant.
- [ ] Logs are useful.
- [ ] Errors are actionable.
- [ ] Handoff is written.
- [ ] Human understood the change.

Close the field manual with a concrete standard readers can adopt.
